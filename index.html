Projects

Print 'N Go

A self-service print vending system powered by a Raspberry Pi and Arduino™ Nano, designed to provide fast and convenient printing for students and professionals. The machine supports Wi-Fi file transfer and USB flash drive uploads, with a coin-based payment system for accessibility in public spaces like schools, libraries, and cafes.

Tech Stack
  • Hardware: Raspberry Pi 4 Model B 2GiB, Arduino™ Nano (Coin Acceptor reader).
  • Backend: Flask (Python web framework), MariaDB.
  • Frontend: HTML, CSS and JS.
  • Additional: Apache HTTP server for reverse proxy.
Features
  • Upload documents via WiFi or USB flash drive.
  • Coin-based automated payment system integrated with Arduino™ Nano.
  • Secure file handling with MariaDB backend.
  • Lightweight Flask web server with integration of Apache HTTP server for reverse proxy.
  • User-friendly web UI for seamless interaction.
